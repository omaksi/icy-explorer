# Icy Explorer

A React-based educational game where players explore an icy world collecting Slovak vocabulary words (vybrané slová po B a M).

## Tech Stack

- Vite + React
- Tailwind CSS
- GitHub Pages for deployment

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Automatic deployment to GitHub Pages via GitHub Actions on push to main branch.

## Commit Style

Use simple one-line commit messages.

## Workflow

Push directly to main, no branching needed.
